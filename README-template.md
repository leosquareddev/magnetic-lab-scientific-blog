## Features

- Display of multiple scientific blog posts.  
- Simulated user login and authentication.  
- Add, edit, and delete comments on posts.  
- Voting system: upvote and downvote comments.  
- Local storage (localStorage) used to save posts, comments, votes, and session.  
- Responsive design for desktop and mobile screens.    

### Built with

- HTML5 - Structure of the blog.
- CSS3, Flexbox, Grid - Styling, animations, and layout.
- JavaScript (ES6+) - DOM manipulation, events, localStorage management, and dynamic content.

### How to use

- Navigate through the site using the main menu: **Home**, **About Us**, **Contact**, and **FAQ** pages.  
- Browse and read multiple blog posts.  
- Filter posts by **discipline**, **title**, or **author** to quickly find content of interest.  
- Add comments to posts (username is required, but **no personal information is needed**).  
- Upvote or downvote comments; votes are saved in localStorage.  
- Log in or log out of the simulated account system.  
- Toggle between **light and dark mode** using the mode switch.  
- Data such as comments, votes, and user session persist locally in your browser thanks to localStorage.


### Privacy Notice

This is a **frontend-only demo project** with simulated user interactivity.  
- No server or database is used; data is stored locally in your browser using localStorage.  
- **Do not enter any real personal information** such as: real names, emails, phone numbers, passwords, or sensitive data.  
- This project is intended for **demonstration and portfolio purposes only**.
